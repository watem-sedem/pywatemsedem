This function uses the rank output raster/dataframe of pywatemsedem.io.modeloutput.identify_rank_sediment_loads() to map a vector point file with information on rank, sediment output.

Parameters:

  • rst_sediexport (str or pathlib.Path | str) – File path of WaTEM/SEDEM sediexport raster.

  • threshold (float) – See pywatemsedem.io.modeloutput.compute_cumulative_loads_in_sinks()

  • vct_out (str or pathlib.Path | str) – File path of output vector.

  • rst_endpoints (str or pathlib.Path | str, default None) – File path to WaTEM/SEDEM sewerin raster (endpoints in pywatemsedem).
